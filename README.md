# 舆情分析专家模板帮助文档

## 1. 模板概述

### 模板简介
舆情分析专家是基于 PydanticAI 框架构建的智能舆情分析系统，能够自动收集、分析和总结网络舆情信息，为企业和组织提供实时的舆情监测和分析服务。

### 适用场景
- 品牌声誉监测与管理
- 危机预警与舆情监控
- 竞品分析与市场调研
- 公众意见收集与分析
- 社交媒体舆情追踪

### 核心功能
- 自动化信息搜集与筛选
- 智能情感分析与分类
- 舆情趋势预测
- 可视化报告生成

## 2. 技术架构

### 使用框架
- **框架**：PydanticAI
- **Agent 类型**：单 Agent 架构
- **核心能力**：集成 LLM、搜索工具、Sandbox 环境

### Agent 架构说明
采用单 Agent 设计，该 Agent 具备多项能力：
- 信息检索能力（通过搜索工具）
- 数据分析能力（通过 Sandbox 环境）
- 自然语言理解与生成（通过 LLM）

### 技术要求
- 支持主流大语言模型（用户可自选）
- 需要配置搜索 API 访问权限
- 需要 Sandbox 环境支持

## 3. 功能说明

### 主要功能模块

**信息采集模块**
- 多源信息抓取（新闻、社交媒体、论坛等）
- 关键词智能匹配
- 实时数据更新

**分析处理模块**
- 情感倾向分析（正面/负面/中性）
- 热点话题提取
- 传播路径追踪
- 影响力评估

**报告生成模块**
- 结构化分析报告
- 数据可视化图表
- 趋势预测建议

### Agent 能力
1. **自主搜索**：根据关键词自动检索相关信息
2. **智能分析**：对海量信息进行情感分析和分类
3. **代码执行**：在 Sandbox 中执行数据处理和可视化代码
4. **报告生成**：自动生成专业的舆情分析报告

### 工作流程
```
用户输入查询 → Agent 执行搜索 → 信息收集 → 
数据清洗 → 情感分析 → Sandbox 数据处理 → 
生成可视化图表 → 输出分析报告
```

## 4. 配置指南

### 在控制台配置

**模型配置**
- 在 AgentRun 控制台选择适合的大语言模型
- 建议使用支持长上下文的模型以处理大量舆情数据
- 可根据分析精度需求选择不同参数的模型

**Sandbox 配置**
- 启用 Sandbox 环境
- 配置允许的 Python 库（如 pandas、matplotlib）
- 设置执行时间和资源限制

## 6. 常见问题

### Q1：搜索结果不准确怎么办？
- 在控制台优化搜索关键词设置
- 调整搜索范围和时间窗口
- 增加搜索深度获取更多结果

### Q2：分析速度较慢？
- 选择更高性能的模型
- 减少搜索结果数量
- 使用快速分析模式

### Q3：如何提高分析准确度？
- 选择专门针对情感分析优化的模型
- 调整情感判定阈值
- 启用深度分析模式

### 注意事项
1. **数据隐私**：确保不分析涉及个人隐私的敏感信息
2. **搜索配额**：注意搜索 API 的调用限制
3. **结果时效性**：舆情信息具有时效性，建议定期更新分析
4. **模型选择**：不同模型对中文情感分析能力差异较大，建议测试后选择
5. **可视化限制**：Sandbox 中的图表生成受资源限制，避免处理超大数据集
