## 技术原理

### V1 架构问题
```
opinion_agent
  └─ data_collector (子 Agent)
      └─ save_search_result() 
          → StateSnapshotEvent ❌ 被吞没，无法传播到前端
```

### V2 架构解决
```
opinion_agent
  └─ save_search_result() (直接工具)
      → StateSnapshotEvent ✅ 直接传播到前端
```

**关键：所有工具直接注册到主 Agent，确保事件实时传播！**

## 常见问题
